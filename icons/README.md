# Icon Placeholder Notes

This directory should contain Chrome extension icons in the following sizes:

- **icon-16.png** - 16x16px (browser toolbar)
- **icon-48.png** - 48x48px (extension management page)
- **icon-128.png** - 128x128px (Chrome Web Store)

## Creating Icons

You can create icons using:

1. **Online Tools**
   - favicon-generator.org
   - icon-converter.com
   - pixlr.com

2. **Design Tools**
   - Figma (free)
   - Photoshop
   - GIMP (free)

3. **AI Generation** (for placeholder)
   - DALL-E
   - Midjourney
   - Local Stable Diffusion

## Icon Design Suggestions

- Book or pages symbol (reading theme)
- Speaker symbol (audio/TTS theme)
- Combination of the two
- Purple/blue gradient (matching UI theme)

## Implementation

1. Create/download your icons
2. Convert to PNG format (if not already)
3. Resize to exact dimensions needed
4. Place in this `icons/` directory
5. Update manifest.json paths if needed

Alternatively, use Material Design Icons or Font Awesome to convert vector designs to PNG.
